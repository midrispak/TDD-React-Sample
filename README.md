## Available Scripts

In the project directory, you can run:

### INSTALL DEPENDENCIES
`yarn install`

### RUN / DEBUG
`yarn start`

Runs the app in the development mode.<br />

### TESTING
`yarn test`

### TESTING WITH CODE COVERAGE
`yarn test --coverage`

### DEPLOYMENT
`yarn build`
